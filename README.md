## PROMPT 1
"Desarrolla una réplica exacta de la pantalla mostrada en la imagen utilizando Flutter con Dart. La interfaz debe incluir los siguientes elementos y características:

Requisitos específicos:
AppBar: Debe contener el título "Crunchyroll El Victor" con estilo personalizado

Esquema de colores: Tema principal en color naranja (#FF5722 o similar), con degradados o variaciones si aparecen en la imagen

Estructura de widgets:

Utiliza Scaffold como widget principal

Implementa ListView o GridView según el diseño mostrado

Incluye tarjetas (Cards) para cada elemento del contenido

Añade imágenes de placeholder (NetworkImage o AssetImage) que representen animes/miniaturas

Detalles visuales:

Respeta las proporciones y distribución de elementos

Incluye textos descriptivos debajo de cada miniatura (títulos, episodios, etc.)

Mantén la jerarquía visual y espaciados consistentes

Funcionalidad básica: Los elementos deben ser interactivos (onTap) con navegación simple a una pantalla de detalles vacía por ahora
